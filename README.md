# ForgeUI One Boiler Plate

**Hardware-proven single-page LVGL v9 UI boilerplate for ESP32-P4 touchscreen projects.**

Built natively using **ESP32-P4 UI Studio** (ForgeUI Studio), this project provides a complete firmware baseline ready to build, flash, modify, and extend for real embedded products.

Current validated hardware:

- Waveshare ESP32-P4-WIFI6-Touch-LCD-7B (1024×600)
- ESP-IDF v5.5.x
- LVGL v9

---

# About This Boiler Plate

This repository is one of the official **ForgeUI One Boiler Plates**.

It showcases the **Arctic Glass** theme—a bright, clean, premium HMI featuring frosted glass styling, soft cyan highlights, and a modern industrial design language suitable for commercial embedded products.

Each boilerplate contains:

- Complete ESP-IDF project structure
- Production-ready ForgeUI One runtime
- Professionally designed single-page HMI
- Hardware-proven runtime configuration
- Clean, zero-bloat starting point for your own applications

Simply clone, build, flash, customise, and create your own embedded product.

# Built With ESP32-P4 UI Studio

This project was visually designed and exported using **ESP32-P4 UI Studio**, the open-source visual development environment for creating modern LVGL interfaces.

ESP32-P4 UI Studio includes:

- Visual drag-and-drop UI Designer
- AI-assisted prompt-to-layout generation
- AI Hero Background generation
- Theme Manager
- Asset Manager
- Icon Browser
- LVGL v9 code generation
- Standalone ESP-IDF project export
- Physical ESP32-P4 deployment pipeline

Development workflow:

```text
ESP32-P4 UI Studio
        │
        ▼
Visual Design
        │
        ▼
AI Assisted Layout
        │
        ▼
ForgeUI One Runtime
        │
        ▼
ESP-IDF Build
        │
        ▼
Physical ESP32-P4 Hardware
```

# About ForgeUI One

ForgeUI One is the embedded runtime powering every exported project generated by ESP32-P4 UI Studio.

It provides:

- ESP32-P4 hardware initialization
- LVGL v9 runtime
- Display driver integration
- GT911 capacitive touch support
- ESP-Hosted WiFi integration
- DS3231 RTC support
- SD Card support
- Audio subsystem support
- Theme engine
- Modular runtime architecture

Generated user interfaces are automatically integrated into the runtime and rendered directly on physical hardware.

# Quick Start

Set target:

```bash
idf.py set-target esp32p4
```

Build:

```bash
idf.py build
```

Flash:

```bash
idf.py flash monitor
```
